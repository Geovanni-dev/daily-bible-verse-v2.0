## 🚀 Acesse o App

**Veja o projeto rodando na Vercel:** [Acessar Luz Diária](https://luz-diaria-weld.vercel.app/)

---

## 🔥 O que tem de novo?

- 📱 **Aplicativo PWA:** Pode ser instalado direto no celular como um app nativo.
- 🌗 **Dark Mode nativo:** Alternância entre tema claro e escuro salvando a sua preferência.
- ❤️ **Favoritos e Histórico:** Sistema de curtidas que salva tudo no navegador via LocalStorage.
- 📸 **Imagens para Redes Sociais:** Gera um card estilizado para compartilhar no WhatsApp ou Instagram.
- 📑 **Navegação dinâmica:** Interface SPA dividida em abas fluídas.
- 📚 **Dataset expandido:** Mais de 300 versículos curados em um arquivo JSON.

---

## 🛠️ Tecnologias Utilizadas

- **JavaScript Vanilla:** Toda a manipulação de DOM e lógica construída do zero.
- **CSS Moderno:** Estilização com CSS Variables e efeito Glassmorphism.
- **APIs Web:** Service Workers e Web Manifest para suporte offline.
- **Lucide Icons:** Biblioteca de ícones vetoriais premium.
